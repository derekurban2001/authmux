# Proflex Roadmap

## Done

- Profile registry (`state.json`)
- Tool adapters (Claude `CLAUDE_CONFIG_DIR`, Codex `CODEX_HOME`)
- Commands: add, list, use, run, rename, remove
- Shim generation (`claude-<profile>`, `codex-<profile>`)
- Zero-dependency CLI with friendly output
- Cross-platform install scripts (bash + PowerShell)
- npm package (`@proflex/cli`)
- Cosign-signed releases

- Shell completions (bash, zsh, fish, PowerShell)
- Profile export/import
- Support for additional AI CLI tools
